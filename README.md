# ProjectArkanoid

A classic Arkanoid (breakout) game built in **C++20** with **SFML 2.5**. The project uses C++20 modules for modular code architecture.

## Features

- Multiple block types (normal, indestructible, bonus)
- Power-ups: extend paddle, multi-ball, speed boost
- Text-based level loader with configurable parameters
- High score system with per-player history
- Pause menu, level selection, and game-over screen
- Background music and sound effects

## Requirements

- **Visual Studio 2022** with C++20 module support
- **SFML 2.5** (included in `External/`)

## Building

Open `ProjectArkanoid.sln` in Visual Studio, set platform to **x64**, and build (**F5**).

## Controls

| Input | Action |
|---|---|
| Arrow keys | Move paddle |
| Escape | Pause |
| Mouse | Navigate menus |
